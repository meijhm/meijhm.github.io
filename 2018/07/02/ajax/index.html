JSON数据格式

概述

  JSON: JavaScript Object Notation（中文名：JavaScript 对象表示法），它和JavaScript对比如下：
1、JSON 中没有变量的概念，没有(不用)声明变量
2、没有末尾的分号
3、对象的属性必须加双引号

数据类型

  与JavaScript对比：
1、简单值：使用与 JavaScript 相同的语法，可以在 JSON 中表示字符串、数值、布尔值和 null。
但 JSON 不支持 JavaScript 中的特殊值 undefined。
2、对象{}：是一组无序的键值对
3、数组[]：表示一组有序的值的列表
